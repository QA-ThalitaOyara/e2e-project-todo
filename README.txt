Pré-requisitos:

    Node.js instalado em sua máquina. Verifique a versão com o comando node -v.
    npm (gerenciador de pacotes JavaScript) instalado. O npm vem junto com o Node.js.

Criando o projeto:

    Crie uma pasta para o seu projeto.
    Navegue até a pasta do projeto no terminal.
    Execute o comando npm init -y para criar um arquivo package.json.
    Instale o Cypress com o comando npm install cypress.

Executando o Cypress:

    Execute o comando 'npx cypress open' para abrir o Cypress Runner.
    O Cypress Runner irá abrir uma interface gráfica onde você pode escrever e executar seus testes.
